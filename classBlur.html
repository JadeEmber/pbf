void Blur::Apply ( const glm::vec2 &  offsetscale,
const GLuint &  weights 
)

Apply blur.

Applies the blur to the currently bound texture and renders the result as a fullscreen quad (i.e. an appropriate framebuffer object must be bound by the caller).

Parameters
offsetscaleScale factor for the texture fetch offsets. For a horizontal blur use (1/inputWidth,0) and for a vertical blur use (0,1/inputHeight).
weightsWeight buffer for the blur. A weight buffer can be generated by ComputeWeights.
